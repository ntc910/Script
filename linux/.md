## Adding Proxy server for apt
```sh
sudo vim /etc/apt/apt.conf.d/90proxy
```
Then add this:
```sh
Acquire::http::proxy "http://107.98.150.183:3333/";
Acquire::https::proxy "http://107.98.150.183:3333/";
Acquire::ftp::proxy "http://107.98.150.183:3333/";
```
