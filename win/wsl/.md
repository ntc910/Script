## Adding Proxy server for apt
```sh
sudo vim /etc/apt/apt.conf.d/90proxy
```
Then add this:
```sh
Acquire::http::proxy "http://172.25.128.1:9999/";
Acquire::https::proxy "http://172.25.128.1:9999/";
Acquire::ftp::proxy "http://172.25.128.1:9999/";
```

## Reset port connection
```sh
netsh interface portproxy delete v4tov4 listenport=3333 listenaddress=0.0.0.0

netsh interface portproxy delete v4tov4 listenport=8080 listenaddress=0.0.0.0
 
netsh interface portproxy add v4tov4 listenport=8080 listenaddress=0.0.0.0 connectport=8080 connectaddress=172.17.0.1

netsh interface portproxy delete v4tov4 listenport=8686 listenaddress=0.0.0.0
netsh interface portproxy add v4tov4 listenport=8686 listenaddress=0.0.0.0 connectport=8686 connectaddress=172.17.0.1
```

## Connect git to proxy server
```sh
git config --global http.proxy $http_proxy
git config --global https.proxy $https_proxy
```
## Disconnect git from proxy server
```sh
git config --global http.proxy ""
git config --global https.proxy ""
# OR
cd /path/to/your/repository
git config --unset http.proxy
git config --unset https.proxy

git config --local --unset http.proxy
git config --local --unset https.proxy
```
