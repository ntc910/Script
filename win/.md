## Listen on port
First, always delete old config
```sh
netsh interface portproxy delete v4tov4 listenport=3333 listenaddress=0.0.0.0
```

```sh
netsh interface portproxy add v4tov4 listenport=3333 listenaddress=0.0.0.0 connectport=3333 connectaddress=127.0.0.1

netsh interface portproxy add v4tov4 listenport=3333 listenaddress=0.0.0.0 connectport=3333 connectaddress=172.17.0.1
```
